= Les possibilités de développement avec Office

# Programme de la session
Macros VBA
VSTO
Web add-in
OpenXML SDK

# Macros VBA
Visual Basic embarqué dans les documents
Facile d'accès (enregistreur de macro)
Problèmes de sécurité
Obsolète depuis 2007

# Développement VSTO
Interractions entre une application .net et MS Office
Ajout de fonctionnalités dans Office (rubans, menus contextuels, events)
Word/Excel : Code embarqué dans le document (non recommandé !)

= Démo
VSTO

# Web add-in
Panneau web dans Office
Développement HTML/JS
Office.js pour interragir avec le document

= Démo
Web add-in

# Déploiement Web add-in
Compatible Windows, Web, Mac, iOS
Manifest XML + site web
Windows Store / Dossier réseau / Office 365

# OpenXML SDK
Dév pour Office sans installer Office
Travail sur la structure XML des docs
NuGet : DocumentFormat.OpenXml
.NET Standard 1.3 

= Démo
OpenXML SDK

# Bonus : Office web apps
Modification des docs dans le navigateur
Natif dans Sharepoint / OneDrive
Documents stockés sur votre serveur

# Web Application Open Platform Interface (WOPI)
API à implémenter sur votre serveur
Récupérer doc, vérouiller, mettre à jour
Utilisable avec Office Online ou Office iOS/Android
